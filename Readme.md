Simple Foundation Theme
============
Based on Foundation 5.5.1


Installation
--------------
#### clone repository

```sh
git clone https://github.com/PatrickTaddey/foundation-theme.git
```

#### Setup client

```sh
cd foundation-theme/
npm install
bower install

# start development in src/dev/* Folders
# output is in src/build/*
grunt build

# start deployment
# output is in src/release/* Folders
grunt release
```
